# Eco Icons

21 hand-crafted line icons for sustainability projects. Free for developers, designers, and anyone.

## Features

- 🌱 **21 Unique Icons** - Carefully designed line icons focused on sustainability themes
- 🎨 **Hand-Crafted** - Each icon is meticulously crafted for clarity and consistency
- 💚 **Free to Use** - Completely free for personal, commercial, and open-source projects
- 📄 **MIT Licensed** - Use with confidence under the MIT License
- 🔧 **Easy to Integrate** - Simple SVG format for seamless integration into any project

## Usage

### Getting Started

1. Clone the repository:
```bash
git clone https://github.com/samnahid/eco.git
```

2. Navigate to the project:
```bash
cd eco
```

3. Use the icons in your project by referencing the SVG files directly or embedding them in your HTML.

### Including Icons in HTML

```html
<img src="path/to/icon.svg" alt="Icon description">
```

Or embed inline:

```html
<svg>...</svg>
```
